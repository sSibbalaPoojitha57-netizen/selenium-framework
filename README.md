# Selenium Automation Framework

A scalable and maintainable Selenium Test Automation Framework built using **Java, Selenium WebDriver, TestNG, Maven, WebDriverManager**, and **Page Object Model (POM)** design pattern.

This framework is designed to support functional UI automation with clean architecture, reusable utilities, and detailed execution reporting.

---

## Key Features
- Selenium WebDriver for browser automation
- TestNG for test execution, assertions, and lifecycle management
- Page Object Model (POM) for maintainable test design
- Extent Reports for detailed HTML test reports
- WebDriverManager for automatic browser driver management
- Utility classes for screenshots and reporting
- TestNG listeners for failure handling
- Maven for dependency management and build execution
